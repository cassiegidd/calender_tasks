# calender_tasks

## Description

This calendar application can be used by anyone who wants to better organize their daily working hours. The application will access the users local storage to set and save their own tasks or reminders for whatever hour it is applicable to them.

## Usage
The user can type their chosen input into the text area and click the save button on the right. The input will then be saved and accessable to the user even after shutting down or refreshing their browser.
